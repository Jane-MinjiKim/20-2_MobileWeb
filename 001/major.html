N분의 1

- 2020년 1학기에 수강한 '모바일 프로그래밍'에서 진행한 팀프로젝트
- 개발언어: Kotlin


What is 'N분의 1'?

- OTT 서비스 사용료를 함께 부담할 사용자들을 쉽게 모을 수 있는 서비스를 제공하는 앱
- 미완성인 채로 프로젝트가 끝나버려서 졸업 전에 완성해보고자 합니다.


담당 파트

- UI/UX Design hard coding
- firebase 연동하여 서버와 database 연결
- 핵심 기능인 게시글 작성 및 실시간으로 정렬된 게시판 구현

홉필드 기반 한글 인식 프로그램

개발 언어: Python
학습 클래스: '가', '녀', '바', '츠', '히' 등
패턴: 7 x 7 패턴


상세 설명

- 2020년 1학기에 수강한 '인공지능'에서 배운 이론을 토대로 구현해본 프로그램입니다.
- 홉필드 모델에서는 '패턴 x 0.15 > 클래스 개수'를 신경써서 설정해야하기 때문에 처음 시도할 때 학습할 클래스를 6개로, 패턴을 7 x 7 = 49개로 진행했었습니다. 공식대로 맞췄지만 제대로 인식이 되지 않아 학습 클래스를 5개로 줄일 수 밖에 없었습니다. 이후에 패턴과 클래스를 점차 늘려갈 때도 학습 클래스를 일부 줄여야했던 점이 아쉬움으로 남았습니다.
