Titanic: Machine Learning from Disaster

타이타닉 탑승자 데이터셋으로부터 생존여부를 예측
개발 언어: Python
개발 툴: Colab
채점 결과: 0.78468 (정확도 약 78% 모델 구현)


상세 설명

- 기계학습 모델에서는 SVM 모델을 제일 좋아해서 SVM 모델을 활용하여 학습하고 싶었는데 데이터를 살펴봤을 때 SVM 모델보다는 랜덤 포레스트가 적절해보여서 랜덤 포레스트 모델로 학습시켰습니다.
첫 도전에서 정확도가 약 75%인 생각보다 너무 낮은 정확도가 나와 당황했었는데 데이터 전처리를 조금 더 꼼꼼히 하고 모델 파라미터 조정을 통해 정확도를 조금이라도 더 올린 78% 모델을 만들 수 있었습니다.
